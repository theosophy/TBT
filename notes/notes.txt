There is an easy way to convert a set of strings into a set of <text> and <view> tags. Use this regex in Notepad++:

    ^(.+)$
    <text>\n<view lang="en">\n\1\n</view>\n</text>
    
After the regex applied, you just need to enclose entire text into <body></body> and add the xml header:

    <?xml version="1.0" encoding="UTF-8"?>
    <body>
    </body>
